## Rule
All terminal recordings and video demos must be ≤30 seconds. Demos sell; tutorials teach. These are demos.

## Why
A 2-minute GIF in a README is a wall. Visitors bounce. Agents loading images waste context. Show the wow moment fast.

## Violation Examples
- A 90-second terminal recording showing full install + config + usage
- A video that spends 20 seconds on pip install before showing the tool
